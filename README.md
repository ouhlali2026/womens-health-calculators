# صحة المرأة – Women's Health Calculators

موقع ثابت (Static) لأدوات صحة المرأة التفاعلية باللغة العربية. يحتوي على حاسبات دقيقة للحمل، الدورة الشهرية، التبويض، BMI، وزن الحمل، ونمو الجنين.

## الموقع مباشر على Vercel

## الأدوات المتوفرة

الحمل (Pregnancy):
- حاسبة موعد الولادة
- حساب أسابيع الحمل
- حاسبة أطفال الأنابيب (IVF)
- حاسبة جنس الجنين (للتسلية فقط)
- حاسبة حمل متقدمة

الدورة والتبويض (Period & Ovulation):
- حاسبة الدورة الشهرية
- حاسبة التبويض
- أيام الأمان
- فترة الخصوبة

الصحة العامة (Health):
- مؤشر كتلة الجسم (BMI)
- السعرات الحرارية للحمل والرضاعة
- زيادة الوزن في الحمل
- مخطط نمو الجنين

المدونة (Blog):
- مقالات توعوية وقائية أسبوعية
- لغة بسيطة
- معلومات من مصادر موثوقة (WHO, ACOG, CDC)
